Example 1

Using all defaults. 
          
            $('#textarea1').limit_text();

Example 2

          
            $('#textarea2').limit_text({ max: 10, strict: true, warning_max: 3 });

Example 3

Allow 10 words maximum. No matter what, the user can't enter more than 10 words. Also, disable the warning styling. 
          
            $('#textarea3').limit_text({ max: 10, type: 'word', strict: true, warning_max: 0 });

Example 4

Use a custom named class for the warning class. 
          
            $('#textarea4').limit_text({ max: 10, type: 'word', warning_class: 'super_warning', warning_max: 5 });

Example 5

Place the description element before the textarea. 
          
            $('#textarea5').limit_text({ placement: 'before' });

Example 6

Uses an already existing element for the description. 
          
            $('#textarea6').limit_text({ description_id: 'textarea6display' });

Example 7

Use an text input field instead of a textarea. 
          
            $('#input7').limit_text({ strict: true, max: 10 });
